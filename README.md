## 📌 Objective
Create an interactive dashboard to visualize and compare 
sales performance between two selected regions using 
the Sample Superstore dataset.

---

## 🛠️ Tools & Dataset
| Tool | Details |
|---|---|
| Tool | Tableau Desktop Public Edition |
| Dataset | Sample Superstore (built-in) |
| Domain | Sales & Business Intelligence |

---

## 📋 Tasks Performed
| # | Task |
|---|---|
| 1 | Loaded Sample Superstore dataset |
| 2 | Created folder using Group By (Customer Name & Order ID) |
| 3 | Created Location Hierarchy (Country → State → City) |
| 4 | Created Primary Region & Secondary Region Parameters |
| 5 | Created Calculated Fields for both regions |
| 6 | Created First Order Date calculated field |
| 7 | Built 12 KPI sheets (6 per region) |
| 8 | Built final dashboard with left-right partition |
